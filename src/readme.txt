Day-30-Axios-User-CURD
This project is a simple React application that demonstrates CRUD (Create, Read, Update, Delete) operations using Axios to interact with 
the JSONPlaceholder API. The application allows you to manage user data by displaying existing data from the API and performing operations 
like adding, updating, and deleting users.

Features
Display Users: Fetch and display a list of users from the API.
Add User: Add a new user to the list.
Update User: Edit existing user details.
Delete User: Remove a user from the list.
Responsive Design: The application is designed to be responsive and works on various screen sizes.

Tech Stack
React.js: Frontend library for building user interfaces.
Axios: Promise-based HTTP client for the browser and Node.js.
JSONPlaceholder API: A fake online REST API for testing and prototyping.
